# vidyo.ai-assignment-videoplayer
Technology stack used :
1) Angular
2) Vite
3) HTML5 Canvas library
4) CSS

How to Run this project in your pc :
1) clone this repository
2) npm install -g @angular/cli
3) npm install
4) npm run build
5) npm run preview
